# Employee Management System

This project is a simple backend application that I developed using Spring Boot as part of my OOP assignment.

##  About the Project

The main idea of this project is to create a system for managing employees.  
It allows you to add, view, update, and delete employees using REST API.

While working on this project, I applied Object-Oriented Programming (OOP) concepts and structured the code in a clean and organized way.

##  Project Structure

The project is divided into layers:

- **Model** → represents data (Employee class)
- **Service** → contains business logic
- **Controller** → handles HTTP requests

This structure makes the project easy to understand and maintain.

##  CRUD Operations

The application supports basic CRUD operations:

- **Create** → add a new employee  
- **Read** → view all employees  
- **Update** → update employee information  
- **Delete** → remove an employee  

### Endpoints:

- `GET /employees`
- `POST /employees`
- `PUT /employees/{id}`
- `DELETE /employees/{id}`

## 🧠 OOP Concepts Used

In this project I used:

- **Encapsulation** → private fields with getters/setters  
- **Classes and Objects**  
- **Layered architecture**  

## 🚀 How to Run the Project

1. Open the project in IntelliJ IDEA  
2. Run `EmployeemanagementApplication`  
3. Open in browser:  
   http://localhost:8080/employees
